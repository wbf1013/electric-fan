# electric-fan

一个智障小工具

电风扇！ 为你的夏日带去清凉！

- [Web Demo](https://fan.faxiaosi.com)

## Feature

### 优势

- 随时随地打开风扇
- 便携
- 低功耗
- 操作简单
- 安装便捷

### 劣势

- 没有风
